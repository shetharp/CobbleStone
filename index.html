How to Use the Gallery

The CobbleStone Gallery Grid offers a semantic and efficient extention to the CobbleStone Grid System for creating photo galleries. The CobbleStone Gallery Grid allows you to create a list of gallery items without having to worry about the column-span limits for a group.

By default, each gallery item occupies one-third of the parent wrapper, which is equivalent to col-4 on the CobbleStone Grid System.

<ul class="gallery">
 <li>
  <a href=""><strong>Image Caption</strong></a>
  <img src="" class="null-img" />
 </li>

 <li>
  <a href=""><strong>Image Caption</strong></a>
  <img src="" class="null-img" />
 </li>
</ul">
